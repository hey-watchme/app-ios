# TestFlight What to Test (外部テスター向けテストガイド)

**バージョン**: 1.0 (Build 1)
**最終更新**: 2025-12-06

---

## 🎯 このアプリについて

WatchMeは、音声録音とAI分析による心理状態・感情・行動パターンの可視化ツールです。
療育・教育・ケアサポートの現場で、観測対象の変化を可視化し、適切なサポートを提供することを目的としています。

---

## ✅ 重点的にテストいただきたい機能

### 1. 初回起動と認証フロー ⭐⭐⭐

**テスト手順**:

1. アプリを初回起動
2. 「はじめる」ボタンをタップ
3. 以下のいずれかでアカウント作成:
   - **匿名アカウント** (最も簡単・推奨)
   - **Googleアカウント**
   - **メールアドレス/パスワード**

**確認ポイント**:

- [ ] ロゴ画面が正常に表示される
- [ ] 「はじめる」「ログイン」ボタンが機能する
- [ ] 匿名アカウント作成が即座に完了する
- [ ] Google認証がスムーズに動作する
- [ ] 初回登録後、サンプルデバイス「5歳男児」が自動追加される

**期待される動作**:
- 認証完了後、自動的にホーム画面（ダッシュボード）に遷移
- サンプルデバイスのデモデータが表示される

---

### 2. 録音機能（コア機能）⭐⭐⭐

**テスト手順**:

1. ホーム画面の録音ボタン（赤い丸ボタン）をタップ
2. マイク権限のリクエストダイアログで「許可」をタップ
3. 10秒以上録音
4. 停止ボタンをタップ

**確認ポイント**:

- [ ] マイク権限リクエストダイアログが適切に表示される
- [ ] 権限説明が日本語で理解しやすい
- [ ] 録音中に音声レベルが視覚的に表示される（波形）
- [ ] 録音時間が正確にカウントされる（00:10, 00:11...）
- [ ] 停止後、自動的にサーバーにアップロードされる
- [ ] アップロード中に進捗バナーが表示される
- [ ] アップロード完了後、「送信完了」バナーが表示される

**期待される動作**:
- 録音ファイルが正常にアップロードされる
- エラーが発生しない

**既知の制限**:
- バックグラウンド録音には対応していません（アプリを閉じると録音が停止します）

---

### 3. AI分析結果の表示 ⭐⭐

**テスト手順**:

1. 録音完了後、約2〜3分待つ（AI分析に時間がかかります）
2. ホーム画面を下にスワイプして更新（プルトゥリフレッシュ）
3. 「最新の分析結果」セクションを確認

**確認ポイント**:

- [ ] 気分スコア（Vibe）が表示される（0〜100の数値）
- [ ] 感情分析結果が表示される（喜び・悲しみ・怒りなど）
- [ ] 行動パターンが表示される（会話・笑い・泣きなど）
- [ ] プッシュ通知で分析完了が通知される（通知許可している場合）

**期待される動作**:
- 録音から2〜3分後、分析結果が自動的にアプリに反映される
- プッシュ通知で完了が通知される

**既知の制限**:
- AI分析には2〜3分かかります（即座には表示されません）

---

### 4. レポート画面 ⭐

**テスト手順**:

1. 画面下部の「レポート」タブをタップ
2. 週次レポートを確認

**確認ポイント**:

- [ ] 気分グラフ（日別の推移）が表示される
- [ ] 週のサマリーが表示される
- [ ] グラフやテキストが読みやすい
- [ ] UI が崩れていない

**期待される動作**:
- 過去1週間のデータが視覚的に表示される

---

### 5. 観測対象画面 ⭐

**テスト手順**:

1. 画面下部の「観測対象」タブをタップ
2. プロフィール情報を確認
3. 「デバイス詳細」ボタンをタップ
4. **「QRコードを表示」ボタンをタップ**（新機能）

**確認ポイント**:

- [ ] 観測対象の基本情報（名前、年齢、性別）が表示される
- [ ] QRコードが正常に生成・表示される
- [ ] QRコード画像を保存・共有できる

**期待される動作**:
- QRコードが即座に生成される
- 共有シートで他のアプリに共有可能

---

## ⚠️ 既知の制限事項

### 録音機能

- **フォアグラウンドのみ**: バックグラウンド録音には対応していません
  - アプリを閉じると録音が停止します

### AI分析

- **処理時間**: 録音後、分析結果が表示されるまで2〜3分かかります
  - サーバー側でAI処理を実行しているため

### サンプルデータ

- **「5歳男児」**: 初回登録時に追加されるデバイスはデモデータです
  - 実際の観測対象ではありません
